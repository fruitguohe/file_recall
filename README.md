# 日复一日（FileRecall）- 使用说明

## 📖 软件介绍

**日复一日**（英文名 **FileRecall**）是一款基于间隔重复算法的智能复习工具，帮助你高效管理和复习学习资料，通过科学的复习间隔提高记忆效果。

### 🔍 核心功能

- **智能间隔重复**：基于1、2、4、7、15、30天的科学间隔，自动安排复习时间
- **文件管理**：支持添加单个文件或整个文件夹进行管理
- **定时提醒**：每天固定时间发送复习通知
- **直观界面**：macOS原生GUI，操作简单直观
- **自动扫描**：定期扫描关联文件夹，自动添加新文件
- **智能过滤**：自动忽略系统文件和临时文件

## 🚀 安装方法

### 方法一：直接运行（开发模式）

1. **确保环境**：需要Python 3.7+和PyObjC库
2. **安装依赖**：
   ```bash
   pip install pyobjc
   ```
3. **运行程序**：
   ```bash
   python learning_review.py
   ```

### 方法二：打包应用（推荐）

1. **安装py2app**：
   ```bash
   pip install py2app
   ```
2. **执行打包**：
   ```bash
   python setup.py py2app
   ```
3. **运行应用**：在`dist`文件夹中找到生成的`.app`文件，双击运行

## 📚 使用指南

### 首次使用

1. **启动应用**：首次运行时，会自动打开设置窗口
2. **添加学习资料**：
   - 点击「添加文件夹」：选择包含学习资料的文件夹
   - 点击「添加文件」：选择单个学习文件
3. **设置提醒时间**：默认每天9:00，可根据个人习惯调整
4. **保存设置**：点击「保存并更新定时」按钮

### 日常使用

1. **接收提醒**：每天设定时间会收到复习通知
2. **查看今日复习**：点击通知或直接打开应用，会显示今日需要复习的文件列表
3. **开始复习**：
   - 双击文件：打开文件并标记为已复习
   - 批量选择：按住Command键选择多个文件，点击「批量打开」
   - 查看位置：点击文件名右侧的放大镜图标，在Finder中显示文件位置
4. **完成复习**：打开文件后，系统会自动更新复习间隔，安排下次复习时间

### 管理设置

1. **修改提醒时间**：在设置窗口中调整小时和分钟，点击「保存并更新定时」
2. **管理文件夹**：在设置窗口中查看、添加或移除关联文件夹
3. **手动运行**：点击「立即运行」按钮，手动触发一次复习检查

## 🎯 功能详解

### 间隔重复算法

软件采用科学的间隔重复算法，根据记忆曲线自动调整复习间隔：
- 第一次复习：1天后
- 第二次复习：2天后
- 第三次复习：4天后
- 第四次复习：7天后
- 第五次复习：15天后
- 第六次及以后：30天后

### 智能过滤系统

自动忽略以下类型的文件：
- 系统文件：.DS_Store、Thumbs.db、desktop.ini
- 临时文件：.tmp、.temp、.part、.crdownload
- 忽略文件夹：.git、node_modules、venv、__pycache__、.cache

### 数据存储

- 所有数据存储在 `~/.learning-review/library.json`
- 包含文件路径、添加时间、最后复习时间、复习间隔等信息
- 数据自动备份，确保学习进度不会丢失

## 💡 使用技巧

1. **批量管理**：将相关学习资料放在同一个文件夹中，一次性添加
2. **定期检查**：每周检查一次设置，确保提醒时间适合自己的学习节奏
3. **合理安排**：开始时添加适量文件，避免复习压力过大
4. **坚持使用**：按照系统安排的时间复习，充分利用间隔重复的优势

## 🔧 故障排除

### 通知不显示
- 检查系统通知设置，确保允许应用发送通知
- 确认定时任务已开启（设置窗口中显示「定时已开启」）

### 应用无法启动
- 检查Python版本是否为3.7或更高
- 确认PyObjC库已正确安装
- 查看 `/tmp/learning_review_error.log` 了解详细错误信息

### 文件不显示在复习列表中
- 检查文件是否被智能过滤系统忽略
- 确认文件路径是否正确
- 尝试重新添加文件或文件夹
